<h1>System Monitoring Task</h1>
<center>
<p>In the monitoring task, you will control a set of <strong>six gauges</strong>.</p>

<p>You will have to correct the system, if necessary, by pressing the corresponding function key: when the green light (key [F5]) goes out; when the red light ([F6]) comes on; or when one of the moving arrows moves to the end of one of the four level gauges (keys [F1], [F2], [F3], [F4]).</p>

</center>


<p><b>Controls:</b></p>



<p>1. To reset one of the four gauges (F1 - F4) to the nominal level  you should use the corresponding Function keys (F1 - F4).</p>

<p>2. To reset the F5 light to green (nominal), use the F5 function key.</p>

<p>3. When the F6 light turns red (alert), use the F6 function key to reset it.</p>
